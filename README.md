## CryptoInfo
### Android app that provides list of crypto assets with their respective price and details using CoinPaprika API

Libraries / technologies used:

• Kotlin

• Clean architecture

• SOLID

• Clean code

• Jetpack Compose

• MVVM

• Kotlin flows

• Kotlin coroutines

• Dependecy injection with koin

• Retrofit for network requests

• Unit testing

• Room for caching
