## CryptoInfo
### Android app that provides list of crypto assets with their respective price and details using CoinPaprika API

Libraries, technologies & patterns used:

• Kotlin

• Clean Architecture

• SOLID

• Clean Code

• Single Source Of Truth (SSOT)

• Jetpack Compose

• MVVM

• Kotlin Flows

• Kotlin Coroutines

• Dependecy injection (Koin)

• Retrofit2

• Unit testing

• Room (caching)
